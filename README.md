## Proyectos recientes

### [BlackSilver](https://github.com/break-dev/BlackSilver) — ERP logístico para minería
`React 19` `Zustand` `Mantine v8` `Laravel`

Sistema de compras y distribución de insumos. El problema central era que el mismo producto llegaba con nombres y unidades distintas dependiendo del proveedor, lo que causaba descuadres de inventario. Resolvimos eso con normalización a unidad base y un Kardex que no se puede editar a posteriori. También tiene generación automática de órdenes de compra a partir de cotizaciones aprobadas.

[Frontend](https://github.com/break-dev/BlackSilver) · [API](https://github.com/break-dev/BlackSilverAPI)

### [TinyFood](https://github.com/break-dev/TinyFood) — App de recetas con IA
`React Native` `Expo Router` `NestJS` `Prisma` `Gemini 2.0` `Supabase`

App móvil que le dices qué tienes en la despensa y te sugiere qué cocinar. El backend usa Gemini 2.0 para interpretar los ingredientes y generar recetas adaptadas a las metas del usuario (calorías, alergias, etc.).

[App](https://github.com/break-dev/TinyFood) · [API](https://github.com/break-dev/TinyFoodAPI)
